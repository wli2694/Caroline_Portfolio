    Timebank App

    Mobile app that allows people to share skills & expertise

    Scope: Final Project for Udacity: Intro to the Design of Everyday THings Course (Online)

    Target Audience: People with Limited Time to Run Errands

    My Role: UX and UI Designer

    Timeline: June - Sept 2015

    Project Brief

    Design a mobile timebank iPhone application that allows people to trade an hour of their skills and expertise for an hour of another personâ€™s skills and expertise.

    This project was a great opportunity to practice skills such as sketching and brainstorming ideas, wireframing, user testing, collecting feedback, creating mock-ups and a prototype using Balsamiq Mockups, Sketch 3, and InvisionApp. Overall it was a fun experience in which I learned a lot.

    Sketching

    First I developed different ideas and sketched conceptual models of how to display more or less time. I also sketched ideas for discoverability within the app. I then sketched wireframes of the user interface for the app as well as user interactions.

    User Testing

    Subsequently, I observed and interviewed 3 people interacting with the sketched wireframes I created. I collected and recorded this feedback.

    Notes: Profile photos will make the app more personal. Have purchases and requests on two separate tabs to be less confusing. Include words below menu icons to make them more clear. Add a confirmation page for completed trades.

    Med-Fi Prototyping Using Balsamiq Mockups

    I improved the wireframes based on the user testing and feedback and built digital mockups using Balsamiq Mockups.

    Hi-Fi Visual Design

    After additional testing and feedback, I created UI designs for the app using Sketch. I also added interactivity to the prototype with InvisionApp.

    Conclusion

    From evaluating my design with others, I learned that people of different generations and knowledge level interpret designs differently. It is important to make graphics and words as clear and concise as possible. In future iterations I would want to add a history function to view previous transactions, a menu for looking at categories of skills, and a menu for sorting skills by date posted, rating, most reviews, and cost.
